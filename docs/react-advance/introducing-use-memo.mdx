# Introducing useMemo

## ⚓ useMemo Hook

可以記憶任何類型的數據。

通常用在重新渲染時，不希望某些複雜的計算再次執行，以優化效能。

```jsx
// 使用 useMemo 來記憶陣列
const sortedList = useMemo(() => {
  console.log('Items sorted');
  // Return what you want to store
  return items.sort((a, b) => a - b);
}, [items]);

// empty dependency => never change
const items = useMemo(() => {
  return [5, 3, 1, 10, 9];
}, []);
```

範例中，useMemo 雖然可以記憶 Array，但同時也佔用了內存，因此只適合用於非常需要性能運算的資料上面。
